# PP02 — Производственная практика (Системное администрирование)

## Описание

Данный репозиторий содержит материалы производственной практики ПП.02 по направлению **«Системное администрирование»**.  
В ходе практики выполнялись задачи по администрированию информационной системы организации **ООО «Автошип Сервис»** (шиномонтажный сервис), автоматизации работы системного администратора и обеспечению стабильной и безопасной работы компьютерной инфраструктуры предприятия.

Практика выполнялась в среде **Windows** с использованием **PowerShell** и средств администрирования операционной системы.

---

## Содержание репозитория

### `docs/`
Документация по практике:
- Отчёт по производственной практике;
- Описание предметной области (шиномонтажный сервис);
- Техническое задание на автоматизацию;
- UML-диаграммы (Use Case, Class, Sequence, Activity, State, Deployment);
- Схема компьютерной сети шиномонтажа;
- Инструкции для персонала (администратор, мастер, кладовщик).

---

### `scripts/`
Скрипты администрирования **PowerShell**, адаптированные под инфраструктуру шиномонтажного сервиса:

| Имя файла | Назначение |
|-----------|------------|
| `backup_tires_db.ps1` | Резервное копирование базы данных учета заказов и хранения шин |
| `restore_tires_db.ps1` | Восстановление БД из резервной копии |
| `create_employee.ps1` | Создание учетной записи сотрудника (мастер, администратор) |
| `disable_employee.ps1` | Блокировка уволенного сотрудника |
| `monitoring_posts.ps1` | Мониторинг состояния рабочих станций на постах шиномонтажа |
| `disk_check_balance.ps1` | Проверка свободного места на сервере и ПК |
| `network_check_kassa.ps1` | Проверка доступности кассовых терминалов и принтеров чеков |
| `update_windows_tires.ps1` | Централизованное обновление Windows на всех ПК сервиса |
| `cleanup_temp.ps1` | Очистка временных файлов и кэша 1С |
| `antivirus_check_tires.ps1` | Проверка статуса антивирусной защиты на постах |
| `logs_analysis.ps1` | Сбор и анализ системных журналов (ошибки, сбои) |
| `printer_check.ps1` | Проверка состояния принтеров этикеток (для маркировки шин) |
| `tire_storage_backup.ps1` | Бэкап фотофиксации шин клиентов (сетевое хранилище) |
| `report_daily.ps1` | Формирование ежедневного отчета о состоянии IT-инфраструктуры |

## Цель работы

Целью производственной практики являлось получение практических навыков системного администрирования, автоматизация задач обслуживания IT-инфраструктуры шиномонтажного сервиса, обеспечение сохранности данных клиентов (учет хранения шин), бесперебойной работы постов и касс, а также повышение надежности работы информационной инфраструктуры предприятия.

---

## Используемые технологии

- **ОС:** Windows 10/11, Windows Server;
- **Скриптинг:** PowerShell 7+;
- **Виртуализация:** Hyper-V (тестовая среда);
- **Моделирование:** UML (PlantUML, Draw.io);
- **СУБД:** PostgreSQL / MySQL (учет заказов);
- **Система контроля версий:** GitHub;
- **Мониторинг:** Zabbix (адаптация под автосервис).
